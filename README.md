# Task Tracker API

Серверное приложение для управления задачами сотрудников.

## Функциональность
- CRUD для сотрудников и задач
- **Занятые сотрудники** — сортировка по количеству активных задач
- **Важные задачи** — задачи, от которых зависят другие, с предложенными исполнителями

## API Endpoints

| Метод | URL | Описание |
|-------|-----|----------|
| GET/POST | /api/employees/ | Список / создание сотрудников |
| GET/PUT/DELETE | /api/employees/{id}/ | Просмотр / обновление / удаление сотрудника |
| GET | /api/employees/busy/ | Занятые сотрудники |
| GET/POST | /api/tasks/ | Список / создание задач |
| GET/PUT/DELETE | /api/tasks/{id}/ | Просмотр / обновление / удаление задачи |
| GET | /api/tasks/important/ | Важные задачи |
